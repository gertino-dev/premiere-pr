# premiere-pr

Un petit dépôt d'exemple pour pratiquer le flux de Pull Request sur GitHub.

## Objectif

Ce repo sert de bac à sable pour apprendre à :
- cloner un dépôt
- créer une branche
- faire un commit
- ouvrir une Pull Request

## Utilisation

Il n'y a rien à installer, ce dépôt est uniquement pédagogique.
